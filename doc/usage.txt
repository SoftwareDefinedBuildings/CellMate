USAGE:
  snaplink [global options] command [command options] [arguments...]

GLOBAL OPTIONS:
  -h, --help      Print this help message
  -v, --version   Print the version number

COMMANDS:
  run             Run the SnapLink server
  convert         Convert database of other formats to SnapLink database
  label           Label a SnapLink database
  show            Visualize a SnapLink database
